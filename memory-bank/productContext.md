# Product Context

## 현재 디자인 프로젝트

메모 앱 UI 디자인 — 개인 메모/할일을 카드 그리드로 관리하는 앱

## 디자인 방향

- 따뜻한 오프화이트 배경 (#F9F7F4)
- 카드별 그라디언트 색상 (노랑/파랑/보라/주황/초록/골드 6색 로테이션)
- pill 형태 탭 네비게이션
- Inter 폰트 패밀리 사용
- 모바일 대응 가능한 auto-layout 기반 구조

## 타겟 사용자

- 디자이너/PM이 빠르게 메모를 분류하고 관리
- 카테고리: 전체, AI Fitness, 상무님 보고, 디자인 업데이트
